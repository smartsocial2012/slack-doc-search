# 📂 Slack Document Explorer & Q&A Bot

이 프로젝트는 슬랙 채널에 흩어져 있는 수많은 파일들을 자동으로 수집하고 인덱싱하여, 사용자가 질문하면 **관련 문서를 즉시 찾아주고 내용을 요약**해주는 스마트 파일 탐색 봇입니다.


## 🌟 주요 특징

**파일 자동 탐색**: 채널에 초대되는 즉시 기존 파일들을 스캔하여 데이터베이스에 등록합니다.

**스마트 검색 & 답변**: 사용자가 질문하면 문서 내용을 분석하여 답변하고, 근거가 된 원본 파일을 채팅창에 직접 업로드해 줍니다.

**강력한 텍스트 추출**: PDF, PPTX, DOCX는 물론, 까다로운 **HWPX(직접 파싱)**와 Excel/CSV 데이터까지 지원합니다.

**보안 유지**: 비공개 채널의 문서는 해당 채널 내에서만 검색되도록 엄격히 구분하여 관리합니다.



## 🛠 필수 준비물

이 봇은 컨테이너 기반으로 동작하므로 다음 파일들이 반드시 필요합니다:

1. **`Dockerfile`**: Python 가상환경(`venv`)을 생성하고 문서 추출에 필요한 시스템 라이브러리를 설치합니다.


2. **`docker-compose.yml`**: 봇 애플리케이션과 벡터 데이터를 저장할 PostgreSQL DB를 연결합니다.


3. **`.env`**: 슬랙 토큰(`SLACK_BOT_TOKEN`, `SLACK_APP_TOKEN`)과 `OPENAI_API_KEY`가 포함되어야 합니다.

## 🚀 설치 및 실행

### 1. 서비스 빌드 및 실행

```bash
docker-compose up --build -d

```

`docker-compose`를 통해 DB 설정 없이 즉시 봇을 가동할 수 있습니다.



### 2. 지원 파일 형식

**문서**: PDF, PPTX, DOCX, HWPX, TXT, MD 

**데이터**: XLSX, CSV (Pandas를 이용한 구조적 추출) 



## 📝 봇 사용법

**파일 학습**: 채널에 파일을 업로드하면 봇이 자동으로 "분석 중" 메시지를 띄우며 학습을 시작합니다.

**파일 찾기 & 질문**: `@봇이름 [질문 내용]`을 입력하세요. 봇이 관련 내용을 설명하고 해당 파일을 찾아줍니다.

**개인 비서**: 봇에게 DM(1:1 메시지)을 보내면 나만의 개인 문서 보관소처럼 사용할 수 있습니다.
